# Monad Agent Skill Files

Static, commit-pinned copies of partner skill manifests used by the Monad Skill Directory.

## Files

| Project | Source | Local path |
| --- | --- | --- |
| Nad.fun | https://nad.fun/skill.md | `skills/nadfun/SKILL.md` |
| DevFun Arena | https://arena.dev.fun/skills/arena.md | `skills/devfun-arena/SKILL.md` |

## Update Process

1. Fetch the latest source file as plain text.
2. Verify the downloaded file matches the live source content.
3. Commit the update.
4. Update the Monad Skill Directory to reference the raw GitHub URL pinned to the new commit SHA.

This repository intentionally has no build step or package manager setup.
